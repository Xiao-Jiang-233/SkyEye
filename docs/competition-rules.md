# 智海算法调优 — 比赛规则摘要

> 来源：2026 睿抗机器人开发者大赛 智能算法赛项规则文件
> 仅保留与训练/推理相关的内容

## 赛题任务

天气图片分类。图片来源于互联网，包括多种拍摄视角及场景。

## 成绩评定

### 评分

省赛评分标准：**最终得分 = Macro F1 × 100**，系统自动评分。

### 排名规则（同分时依次比较）

1. 模型推理时间由短到长排序
2. 代码执行效率（内存占用率、CPU/GPU 利用率）
3. 代码规范性（注释完整性、代码结构清晰度）

### 违规

抄袭或影响比赛公平性的行为直接取消资格。禁止攻击系统或窃取数据，组委会保留对异常提交的审查权。

## 关键约束

- 推理使用 **CPU**
- 推理总时限 ≤ **70 分钟**（整体，非单张）
- 提交：推理代码 + 模型权重
- 可用预训练模型，可安装第三方库
- 禁止外接大模型 API
