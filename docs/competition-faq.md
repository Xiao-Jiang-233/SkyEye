# 智海算法调优 — FAQ（训练/推理相关）

> 来源：2026 睿抗 FAQ（更新至 6.3），仅保留与训练/推理相关的条目

## 比赛机制

**Q: 比赛结果是只有一次机会还是能多次提交？**
A: 测试无限制次数，**提交只有一次**。比赛分为训练集、测试集、评分集。选手只能拿到训练集，测试集用于代码测试评估模型效果。最终分数为提交的模型在评分集上的推理分数。

**Q: 三天包括自己训练模型吗？**
A: 包括。

**Q: 比赛模式是类似 Kaggle 那种实时动态排榜吗？**
A: 类似 Kaggle 调优提交，但**没有实时动态榜**。

## 数据与类别

**Q: 比赛规则中有给出具体要分类的类别吗？**
A: **多云、雨天、雪天、晴天**（4 类）。（注：MO 平台测试集实际含 6 类，非官方。）

**Q: 数据集中的图片是卫星云图还是手机拍的照片？**
A: 类似**手机拍摄**的图片。

**Q: 图片格式？像素大小？需不需要预处理？**
A: 像素大小**没有统一标准**，需要数据预处理。

**Q: 用来评分的数据集大概有多少张？**
A: **几千张**。

**Q: 需要自己标注数据吗？**
A: 不需要。

**Q: 预选赛素材（数据集）什么时候提供？**
A: **开赛提供**。

## 训练

**Q: 必须在平台上训练吗？还是本地也可以？**
A: **可以本地训练**后将模型及代码传到平台上；可以安装其他库。

**Q: MO 上边的 GPU 算力是无限的吗？**
A: GPU 资源有限，过多人使用需要排队。GPU 时间可以加。

**Q: 深度学习用什么框架？**
A: 推荐 **PyTorch**。

**Q: 可以用开源模型吗？预训练权重呢？**
A: 可以用开源模型，可以使用预训练模型。

**Q: 可以用大模型 API 吗？**
A: **外接大模型 API 不可以**，仅限深度学习模型。使用大模型辅助编程可以。

## 推理与提交

**Q: 最终推理时给什么配置？**
A: 推理使用 **CPU**。

**Q: 比赛要求时间是单张还是整体？**
A: **整体时间**。

**Q: 提交代码后推理时限？**
A: CPU 测试，测试时长应在 **70 分钟**内。

**Q: 提交什么内容？**
A: 提交**推理代码和权重**。

**Q: 会提供基线吗？**
A: 基线在答题界面中提供，不需要下载。具体提供哪些暂不告知。

## 评分

**Q: F1 分数是 Macro F1 还是 Micro F1？**
A: **Macro F1**。
