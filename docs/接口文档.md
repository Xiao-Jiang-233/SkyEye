## config — 全局配置

### `config.CONFIG`

超参数字典，所有模块通过 `from config import CONFIG` 统一获取。

```python
from config import CONFIG

# 读取配置
device = CONFIG["device"]          # "cuda" | "cpu"
batch_size = CONFIG["batch_size"]  # 32
```

| 键 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `device` | `str` | `"cuda"` | 训练设备（自动检测） |
| `seed` | `int` | `42` | 随机种子 |
| `fp16` | `bool` | `True` | 混合精度训练（仅 CUDA 生效） |
| `num_workers` | `int` | `0`（Win）/ `4` | DataLoader 线程数 |
| `teacher_model` | `str` | `"efficientnet_b5"` | timm 教师模型名 |
| `teacher_epochs` | `int` | `10` | 教师训练轮数 |
| `student_model` | `str` | `"efficientnet_b0"` | timm 学生模型名 |
| `kd_temperature` | `float` | `4.0` | 蒸馏温度 |
| `kd_alpha` | `float` | `0.7` | 软标签损失权重 |
| `prune_ratio` | `float` | `0.4` | 最终剪枝比例 |
| `prune_iterations` | `int` | `2` | 渐进剪枝轮数 |

> 完整配置项见 [config.py](../config.py)。

## data — 数据加载与增强

### `get_train_transforms(img_size=224) → transforms.Compose`

训练集数据增强：RandomResizedCrop + RandAugment + 归一化。

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `img_size` | `int` | `224` | 输入尺寸 |

### `get_val_transforms(img_size=224) → transforms.Compose`

验证集数据增强：Resize + CenterCrop + 归一化。

### `prepare_data(data_sources, writable_root, class_names) → str`

将多个数据源合并到可写目录（仅首次运行）。支持平铺/嵌套型目录结构、.zip 自动解压、类名自动映射。

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `data_sources` | `list[dict]` | `[{"path": "...", "class_map": {...}}, ...]` |
| `writable_root` | `str` | 目标可写目录 |
| `class_names` | `list[str]` | 标准类别名 |

**返回**: `str` — 合并后数据目录路径。

### `create_dataloaders(data_root, img_size, batch_size, num_workers) → tuple`

创建训练/验证 DataLoader，内部做分层划分（stratified split）。

**返回**: `(train_loader, val_loader, class_counts)`

- `train_loader: DataLoader` — 训练集，shuffle=True
- `val_loader: DataLoader` — 验证集，shuffle=False
- `class_counts: np.ndarray` — 各类别样本数

### `compute_class_weights(class_counts) → torch.Tensor`

根据类别样本数计算 FocalLoss 的 alpha 权重（反比归一化）。

## models — 模型定义与蒸馏

### `WeatherEfficientNet(model_name, num_classes, pretrained) → nn.Module`

EfficientNet 天气分类器。基于 timm，替换分类头，通过 forward hook 捕获中间层特征。

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `model_name` | `str` | `"efficientnet_b0"` | timm 模型名 |
| `num_classes` | `int` | `9` | 类别数 |
| `pretrained` | `bool` | `True` | 是否加载预训练权重（走 HF 镜像） |

```python
model = WeatherEfficientNet("efficientnet_b5", num_classes=6, pretrained=True)

# 普通前向
logits = model(images)                            # (B, 6)

# 带中间层特征（知识蒸馏时使用）
logits, feats = model(images, return_features=True)  # feats: {"stage_0": ..., ...}
```

**方法**:

| 方法 | 返回 | 说明 |
| --- | --- | --- |
| `forward(x, return_features=False)` | `Tensor` 或 `(Tensor, dict)` | 前向传播 |
| `get_stage_channels()` | `list[int]` | 各 stage 输出通道数（去重） |

### `FeatureProjector(student_channels, teacher_channels) → nn.Module`

1×1 卷积投影层，将 Student 特征通道映射到 Teacher 通道（用于中间层特征蒸馏）。

### `DistillationTrainer(teacher, student, device, cfg)`

知识蒸馏训练器。组合损失：`α·KL(soft_S, soft_T)·T² + (1-α)·CE(S, labels) + β·MSE(feat_S, feat_T)`

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `teacher` | `nn.Module` | 教师模型（将被冻结） |
| `student` | `nn.Module` | 学生模型 |
| `device` | `torch.device` | 训练设备 |
| `cfg` | `dict` | CONFIG 字典 |

**方法**:

| 方法 | 返回 | 说明 |
| --- | --- | --- |
| `train(train_loader, val_loader)` | `nn.Module` | 完整蒸馏训练，返回最佳学生模型 |
| `train_step(images, labels, optimizer, scaler)` | `(float, float, float)` | 单步训练，返回 (total, kd, feat) loss |
| `evaluate(loader)` | `(float, float)` | 验证，返回 (f1, acc) |

## training — 训练流程

### `train_teacher() → nn.Module`

训练教师模型（EfficientNet-B5）。FocalLoss + 混合精度 + CosineAnnealingLR。

**输出**: `results/teacher_best.pth`

### `run_distillation() → nn.Module`

执行知识蒸馏 (B5 → B0)。加载教师 → 创建学生 → DistillationTrainer.train()。

**输出**: `results/student_distilled_best.pth`

### `prune_and_finetune() → nn.Module`

结构化剪枝 + 渐进微调。2 轮渐进剪枝 (20% → 40%)，每轮后微调，最终固化 mask。

**输出**: `results/student_pruned_final.pth`

### `StructuredPruner(model, prune_ratio, method)`

结构化通道剪枝器，对 MBConv 1×1 点卷积做 L2-norm 剪枝。

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `model` | `nn.Module` | — | 待剪枝模型 |
| `prune_ratio` | `float` | `0.4` | 剪枝比例 |
| `method` | `str` | `"l2"` | 剪枝准则 (`"l2"` / `"l1"`) |

### `finetune_after_prune(model, train_loader, val_loader, device, cfg, epochs, lr, tag) → nn.Module`

剪枝后微调。

## inference — 推理与导出

### `export_to_onnx(model_path, onnx_path=None) → str`

将 PyTorch 模型导出为 ONNX FP32 格式（opset 13，支持动态 batch）。

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `model_path` | `str` | `.pth` 权重文件路径 |
| `onnx_path` | `str` | 输出路径（默认 `results/weather_model.onnx`） |

**返回**: `str` — ONNX 文件路径。

### `quantize_to_int8(onnx_path=None, int8_path=None) → str`

对 FP32 ONNX 模型进行 INT8 动态量化（CPU 推理加速 2-4×）。

**返回**: `str` — INT8 ONNX 文件路径。

### `benchmark_cpu(onnx_path, int8_path, warmup=20, runs=100) → dict`

CPU 推理测速，对比 FP32 vs INT8。

**返回**: `{"fp32_ms": float, "int8_ms": float, "speedup": float}`

### `predict_image(image_path, model=None, device=None) → dict`

单张图片 PyTorch 推理。

**返回**:
```python
{
    "prediction": "sunny",       # 预测类别
    "confidence": 0.9532,        # 置信度
    "top_k": [
        ("sunny", 0.9532),       # Top-3 结果
        ("cloudy", 0.0311),
        ("rainy", 0.0086),
    ]
}
```

### `predict_batch(image_paths, model=None, device=None) → list[dict]`

批量推理，返回每张图片的预测结果。

### `load_model(model_path=None, device=None) → (nn.Module, str)`

加载训练好的 PyTorch 模型。

## utils — 工具函数

### `FocalLoss(alpha, gamma, reduction) → nn.Module`

Focal Loss 损失函数：`FL = -(1-pt)^γ · log(pt)`

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `alpha` | `Tensor` | `None` | 各类别权重 |
| `gamma` | `float` | `2.0` | 聚焦参数 |
| `reduction` | `str` | `"mean"` | `"mean"` / `"sum"` |

### `evaluate_model(model, loader, device) → (float, float)`

验证集评估，返回 `(macro_f1, accuracy)`。

### `compute_metrics(all_labels, all_preds, class_names) → dict`

计算完整分类指标。

**返回**: `{"f1": ..., "accuracy": ..., "confusion_matrix": ..., "report": ...}`

### `print_metrics(metrics) → None`

格式化打印评估指标（F1 / Accuracy / 分类报告 / 混淆矩阵）。

### `TrainLogger(log_dir=None, use_tb=True)`

训练日志记录器，支持 TensorBoard 可视化。

各训练阶段自动创建独立日志目录：

| 阶段 | 日志目录 |
|------|----------|
| 教师训练 | `results/tb_results/teacher/` |
| 知识蒸馏 | `results/tb_results/distill/` |
| 剪枝微调 | `results/tb_results/prune_iter1/`, `prune_iter2/`, `prune_final/` |

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `log_dir` | `str` | 自动生成时间戳目录 | TensorBoard 日志目录 |
| `use_tb` | `bool` | `True` | 是否启用 TensorBoard |

**方法**:

| 方法 | 说明 |
| --- | --- |
| `log_scalar(tag, value, step)` | TensorBoard 标量记录 |
| `log_metrics(phase, metrics, epoch)` | 记录一个 epoch 的指标集，自动展开为 `{phase}/{name}` 标量 |
| `elapsed()` | 返回已用时间（秒） |
| `close()` | 关闭 TensorBoard writer |

**使用示例**:

```python
from utils.logger import TrainLogger

logger = TrainLogger(log_dir="results/tb_results/my_run", use_tb=True)

# 每个 epoch 记录一次
logger.log_metrics("train", {"loss": 0.35}, epoch=1)
logger.log_metrics("val", {"f1": 0.85, "acc": 92.3}, epoch=1)

logger.close()
```

**启动 TensorBoard 查看**:

```bash
tensorboard --logdir results/tb_results/
```

浏览器访问 `http://localhost:6006`，在 **SCALARS** 页查看 loss 下降曲线和 F1/Accuracy 上升曲线。
