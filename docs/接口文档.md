## config — 全局配置

### `config.CONFIG`

超参数字典，所有模块通过 `from config import CONFIG` 统一获取。

```python
from config import CONFIG

device = CONFIG["device"]          # "cuda" | "cpu"
batch_size = CONFIG["batch_size"]  # 8
```

### 数据

| 键 | 类型 | 值 | 说明 |
| --- | --- | --- | --- |
| `data_roots` | `str` | `"auto"` | 自动扫描 datasets/ |
| `writable_root` | `str` | `"_data/weather"` | 数据合并可写目录 |
| `class_names` | `list` | 7 类（已 `sorted` 匹配 ImageFolder） | 所有已知类 |
| `skip_classes` | `list` | `["other"]` | 暂不训练的类 |
| `active_class_names` | `list` | 自动推导（6 类，去 `other`） | 模型实际输出类（推理/ONNX/bias 必须用） |
| `num_classes` | `int` | `len(active_class_names)` (=6) | 实际训练类数 |
| `class_aliases` | `dict` | haze→foggy, snow→snowy, dew→other, … | 类名映射表 |
| `img_size` | `int` | `380` | EfficientNet-B4 原生分辨率 |
| `batch_size` | `int` | 自适应（~1GB/样本） | 按 VRAM 自动计算 |
| `val_split` | `float` | `0.15` | 验证集比例 |

### 教师模型

| `teacher_weight_decay` | `float` | `1e-4` | AdamW 正则化 |
| `warmup_epochs` | `int` | `2` | Linear warmup |

### 知识蒸馏

| `kd_feature_weight` | `float` | `0.1` | 中间层特征损失权重 |
| `kd_epochs` | `int` | `15` | 蒸馏轮数 |
| `kd_lr` | `float` | `1e-3` | 学生训练 LR |

### 训练策略

| 键 | 类型 | 值 | 说明 |
| --- | --- | --- | --- |
| `use_focal_loss` | `bool` | `True` | 启用 FocalLoss |
| `focal_gamma` | `float` | `1.0` | 聚焦参数 |
| `label_smoothing` | `float` | `0.0` | 关闭（保护 KD 软标签） |
| `per_class_label_smoothing` | `dict` | `sunny/cloudy ε=0.1` | 方案 D：对易混淆类更高平滑 |
| `mixup_alpha` | `float` | `0.2` | MixUp 混合强度 |
| `ema_decay` | `float` | `0.99997` | EMA 衰减率 |
| `scheduler` | `str` | `"cosine"` | LR 调度策略 |
| `logit_bias` | `dict` | `cloudy=0.55, foggy=-0.25` | 方案 A：判定门槛调整（公式 `logits - bias`，bias>0→更难被判为该类） |
| `confusion_penalty_weight` | `float` | `0.3` | 方案 B：sunny/rainy/foggy→cloudy 三方向额外惩罚 |

## data — 数据加载与增强

### `get_train_transforms(img_size=380) → transforms.Compose`

训练集数据增强：RandomResizedCrop + RandAugment + 归一化。

### `get_val_transforms(img_size=380) → transforms.Compose`

验证集数据增强：Resize + CenterCrop + 归一化。

### `prepare_data() → str`

将多数据源合并到可写目录（仅首次运行，后续直接返回路径）。自动处理平铺/嵌套目录、.zip 解压、类名映射、只读限制。

**返回**: 合并后数据目录路径。

### `create_dataloaders(data_root, img_size, batch_size, num_workers, cloudy_oversample, sunny_oversample) → tuple`

创建训练/验证 DataLoader，内部做分层划分。

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `data_root` | `str` | 自动 | 数据目录 |
| `img_size` | `int` | CONFIG | 输入尺寸 |
| `batch_size` | `int` | CONFIG | 批次大小 |
| `num_workers` | `int` | CONFIG | Worker 线程数（传 0 不会被 or 覆盖） |
| `cloudy_oversample` | `bool` | `False` | cloudy 过采样 2×（已关闭，精度低） |
| `sunny_oversample` | `bool` | `False` | sunny 过采样 2×（P2 阶段开启） |

**返回**: `(train_loader, val_loader, class_counts, class_names)`

### `compute_class_weights(class_counts) → torch.Tensor`

根据类别样本数反比归一化，计算 FocalLoss alpha。

## models — 模型定义与蒸馏

### `WeatherEfficientNet(model_name, num_classes, pretrained) → nn.Module`

EfficientNet 天气分类器。基于 timm，替换分类头，通过 forward hook 捕获中间层特征。

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `model_name` | `str` | — | timm 模型名（B4/B0） |
| `num_classes` | `int` | — | 类别数（教师 6） |
| `pretrained` | `bool` | `True` | 预训练权重（HF 镜像） |

```python
model = WeatherEfficientNet("efficientnet_b4", num_classes=6, pretrained=True)

# 普通前向
logits = model(images)                              # (B, 6)

# 带中间层特征（蒸馏时使用）
logits, feats = model(images, return_features=True)  # feats: {"stage_0": ..., ...}
```

**方法**:

| 方法 | 返回 | 说明 |
| --- | --- | --- |
| `forward(x, return_features=False)` | `Tensor` 或 `(Tensor, dict)` | 前向传播 |
| `get_stage_channels()` | `list[int]` | 各 stage 输出通道数 |

### `FeatureProjector(student_channels, teacher_channels) → nn.Module`

1×1 卷积投影层，将 Student 特征通道映射到 Teacher 通道。

### `DistillationTrainer(teacher, student, device, cfg)`

知识蒸馏训练器。损失 = α·KL(soft_S, soft_T)·T² + (1-α)·CE(S, labels) + β·MSE(feat_S, feat_T)

| 方法 | 返回 | 说明 |
| --- | --- | --- |
| `train(train_loader, val_loader)` | `nn.Module` | 完整蒸馏训练 |
| `train_step(images, labels, optimizer)` | `(float, float, float, Tensor)` | 单步训练，额外返回 preds 避免双前向 |
| `evaluate(loader)` | `(float, float)` | 验证 (f1, acc) |

## training — 训练流程

### `train_teacher() → nn.Module`

训练教师模型（EfficientNet-B4, 380px）。组合策略：FocalLoss + MixUp + EMA + BF16 + DRW + ConfusionPenalty + LogitAdj。

**策略时间线**:
```
Phase 1 (Epoch 1-9):  Fast+MU      (MixUp α=0.2，标准采样，EMA)
Phase 2 (Epoch 10-15): Fast+OS+MU+CP (sunny 2× 过采样，ConfusionPenaltyLoss，LR 减半)
```

**输出**: `results/teacher_best.pth` + `results/checkpoints/teacher_epoch_XX.pth`

### 辅助模块

#### `mixup_data(x, y, alpha=0.2) → (mixed_x, y_a, y_b, lam)`

MixUp 数据增强 (Zhang et al., ICLR 2018)。

#### `FocalLoss(alpha, gamma, label_smoothing, reduction) → nn.Module`

Focal Loss: `FL = -(1-pt)^γ · CE_smoothed`，支持 label smoothing。

#### `EMA(model, decay=0.999)`

模型权重指数滑动平均。

| 方法 | 说明 |
| --- | --- |
| `update(model)` | 更新 shadow 权重 |
| `apply_shadow(model)` | 替换为 EMA 权重（验证前） |
| `restore(model)` | 恢复训练权重（验证后） |

#### `evaluate(model, loader, device, class_names) → (f1, acc, per_class_f1)`

验证集评估，返回 macro F1、准确率、逐类 F1。使用 `logits - bias` 调整判定门槛。

### `run_distillation() → nn.Module`

知识蒸馏 (B4 → B0)。加载教师 → 创建学生 → DistillationTrainer.train()。

**输出**: `results/student_distilled_best.pth`

## inference — 推理与导出

### `LogitBiasWrapper(model, bias) → nn.Module`

将 logit bias 烘焙到模型图中（`logits - bias`），确保 ONNX 导出包含 bias 逻辑。

### `export_to_onnx(model_path, onnx_path=None) → str`

PyTorch → ONNX FP32（opset 17，动态 batch）。自动调用 `LogitBiasWrapper` 烘焙 bias。

### `quantize_to_int8(onnx_path=None, int8_path=None) → str`

FP32 ONNX → INT8 动态量化（CPU 加速 2-4×）。

### `benchmark_cpu(onnx_path, int8_path, warmup=20, runs=100) → dict`

CPU 推理测速。返回 `{"fp32_ms": float, "int8_ms": float, "speedup": float}`

### `load_model(model_path=None, device=None) → (predictor, device)`

加载推理模型。优先级：INT8 ONNX → FP32 ONNX → PyTorch。显式传 `model_path` 时跳过 ONNX 直接走 PyTorch。

### `ONNXPredictor(session)` / `PyTorchPredictor(module, device, bias)`

统一推理接口（`predictor(tensor) → logits`），logit bias 已内置。

### `predict_image(image_path, model=None, device=None) → dict`

单张图片推理。返回 `{"prediction": str, "confidence": float, "top_k": [(str, float), ...]}`

### `predict_batch(image_paths, model=None, device=None) → list[dict]`

批量推理。

## utils — 工具函数

### `evaluate_model(model, loader, device) → (float, float)`

验证集评估，返回 `(macro_f1, accuracy)`。

### `compute_metrics(all_labels, all_preds, class_names) → dict`

计算完整分类指标：`{"f1": ..., "accuracy": ..., "confusion_matrix": ..., "report": ...}`

### `TrainLogger(log_dir=None, use_tb=True)`

训练日志记录器，支持 TensorBoard SCALARS。

| 方法 | 说明 |
| --- | --- |
| `log_scalar(tag, value, step)` | TensorBoard 标量记录 |
| `log_metrics(phase, metrics, epoch)` | 记录 epoch 指标集 |
| `elapsed()` | 已用时间（秒） |
| `close()` | 关闭 writer |

各阶段自动创建独立日志目录：

| 阶段 | 日志目录 |
|------|----------|
| 教师训练 | `results/tb_results/teacher/` |
| 知识蒸馏 | `results/tb_results/distill/` |
